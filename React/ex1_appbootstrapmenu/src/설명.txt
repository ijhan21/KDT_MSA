container 클래스:

App.js에서 <div> 요소에 container 클래스를 추가하여 Bootstrap의 레이아웃을 적용했습니다. mt-5는 상단 마진을 추가하여 레이아웃에 여백을 제공합니다.
text-center 클래스:

제목을 중앙 정렬하기 위해 text-center 클래스를 사용했습니다.
form-control 클래스:

TodoForm.js에서 입력 필드에 form-control 클래스를 사용하여 Bootstrap의 스타일이 적용된 텍스트 입력 필드를 만들었습니다.
btn 클래스:

버튼에 btn 클래스를 적용하여 기본 Bootstrap 버튼 스타일을 사용했습니다. btn-primary는 파란색 버튼을 나타내며, btn-warning과 btn-danger는 각각 노란색(수정 버튼)과 빨간색(삭제 버튼) 버튼을 나타냅니다.
list-group 클래스:

TodoList.js에서 리스트를 스타일링하기 위해 list-group 클래스를 사용했습니다. 이는 리스트 항목을 Bootstrap 스타일로 꾸며줍니다.
list-group-item 클래스:

개별 리스트 항목에 list-group-item 클래스를 적용하여 Bootstrap 스타일을 적용했습니다. d-flex, justify-content-between, align-items-center 클래스를 추가하여 플렉스박스를 사용해 요소를 정렬하고 배치했습니다
